Un predictor de texto (como el de los motores de busqueda) pero solo para una palabra.
El idioma de las palabras a predecir se pueden cambiar ingresando un diccionario distinto.
Para el proposito de este proyecto se utilizo un diccionario en ingles.

se ejecuta entregandole:
- el ejecutable,
- el diccionario "dic.txt",
- un .txt (ej: quer.txt) con palabras (o palabra) en una linea cada una (estas son las palabras a predecir),
- un .txt (ej: res.txt) que servira para guardar las predicciones que se encontraron para cada palabra en "quer.txt" (no es necesario que el archivo este previamente creado) y
- un numero entero que indica la cantidad de palabras parecidas que se quieren encontrar.
